# Zera's Tools

Zera's Tools is a lightweight, modern desktop application built with WPF (.NET Framework 4.8 / C# 7.3) for converting images to ICO files. It features a clean UI with a sidebar for tool navigation, a home page with links, and customizable settings.

---

## Features

- **Home Page**: Provides information and a link to the GitHub repository.  
- **Image to ICO Converter**: Convert PNG or JPG images into Windows-compatible ICO files. Automatically names the output as `original filename {Converted by Zera}.ico`.  
- **Settings Window**: Access application settings (top-right gear icon).  
- **Modern UI**: Rounded sidebar, draggable top bar, and integrated minimize and close buttons.  
- **Resizable Buttons**: Sidebar buttons adjust size to emphasize important tools.

## Usage
Launch the application. The default page is Home.

Navigate to Image to ICO from the sidebar.

Drag and drop an image or use the Browse button to select a PNG/JPG file.

Click Convert to generate the ICO file.

The output file is saved in the same folder as the original with the name format: `original filename {Converted by Zera}.ico`
